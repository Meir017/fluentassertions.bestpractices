## Install

using the latest stable version:

```powershell
Install-Package FluentAssertions.Analyzers
```

## Getting Started

### Build

```bash
dotnet build src
```

### Tests

```bash
dotnet test src --configuration Release  --filter 'TestCategory=Completed'
```
